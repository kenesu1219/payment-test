Kindly have these on your device:
- PYTHON 3.11.4
- LIBRARIES 
    * ROBOT FRAMEWORK 3.2.2
- DRIVER for Chrome

=================================================
INSTALL PYTHON 3.11.4
=================================================
1. Download Python version 3.11.4
2. Proceed with Installation
3. On search bar, search for environment variables
4. Select edit system environment variables
5. Click environment variables
6. Under user variables, select Path and click edit
7. Add path where Python is installed
8. Click okay
9. Under system variables, select Path and click edit
10. Add path where Python is installed
11. Click okay
12. Open Command Line
13. Type "python --version" and hit enter
14. Python 3.11.4 shall be displayed

=================================================
INSTALL ROBOT FRAMEWORK 3.2.2
=================================================
1. Open commandline
2. Type python -m pip install robotframework==3.2.2 and hit enter
3. Check version by typing python -m pip list
4. Must include robot framework 3.2.2 in the list

=================================================
DRIVER for chrome
=================================================
1. Update your chrome
2. Go to chrome driver
3. Download the latest version that is same as your device
4. Extract the zip file
5. Copy the driver
6. Go to directory where python is installed
7. Go to \Scripts
8. Paste it there
9. On search bar, search for environment variables
10. Select edit system environment variables
11. Click environment variables
12. Under user variables, select Path and click edit
13. Add path where Python is installed
14. Click okay
15. Under system variables, select Path and click edit
16. Add path where Python is installed
17. Click okay
18. Open Command Line
19. Type "python --version" and hit enter
20. Python 3.11.4 shall be displayed
